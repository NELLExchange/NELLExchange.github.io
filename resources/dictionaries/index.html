LEXIN

nynorsk yes

examples yes
inflections yes

A set of dictionaries designed for minority language speakers in Norway to help with learning everyday Norwegian.

The Nynorsk-Bokmål and English-Nynorsk dictionaries can be used together to get English-Bokmål translations.

https://lexin.oslomet.no/

The Norwegian Academy's dictionary.

Contains a small sub-set of 'moderat' bokmål and riksmål (not official).

This makes this dictionary better at handling slang.

https://www.naob.no/ordbok/
